# atliq-hospitality-powerbi
Power BI dashboard analyzing ₹1.7B+ hotel revenue, occupancy, ADR, RevPAR, and booking trends for AtliQ Hotels.
Revenue Insights in Hospitality Domain – AtliQ Hotels
Project Description

This project analyzes the business performance of AtliQ Hotels across four major cities — Mumbai, Bangalore, Hyderabad, and Delhi — using an interactive Power BI dashboard. The objective was to track revenue, occupancy, customer ratings, booking platforms, and pricing metrics to help management make data-driven decisions.

What I Analyzed

Using hotel booking, property, room, and calendar data, I built key performance indicators including:

Total Revenue: ₹1.71 Billion

Occupancy Rate: 57.9%

Average Daily Rate (ADR): ₹12.70K

RevPAR: ₹7.35K

Realisation %: 70.15%

DSRN: 33,225
